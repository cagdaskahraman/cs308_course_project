# Frontend Setup

This frontend uses React + TypeScript + Bootstrap with Vite.

## Run locally

```bash
npm install
npm run dev
```

App URL: `http://localhost:5173`

## Backend API

The app reads products from:

- `http://localhost:3000/products`

To change API base URL:

```bash
VITE_API_BASE_URL=http://localhost:3000
```
